PI超级计算使用常用shell命令

SSH to remote supercomputer and input password

ssh -p [portnumber] [usrname]@[ipaddress]

Use python in supercomputer

Refer to python module

module load ~/python27-gcc/bin/activate

Use python environment

source ~/python27-gcc/bin/activate

Transfer files between local computer and remote computer

scp [filepath in local computer] [usrname]@[ipaddress]:[filepath in remote computer]

Run shell in supercomputer

sbatch [shell file]

Execute task

sbatch -p cpu [slurm file]

Cancel a task

scancel [jobnumber]

Show our task running on supercomputer

squeue -u whoami
